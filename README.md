# Calculadora de IMC 

## Descrição 

Este projeto é uma aplicação web simples para calcular o **Índice de Massa Corporal (IMC)**. O usuário insere o peso e a altura, e a aplicação retorna o IMC e a classificação correspondente, conforme a tabela de referência. 
Projeto configurado para ser executado dentro de um **container Docker**, facilitando o deploy e garantindo que funcione em qualquer máquina com Docker instalado.

--- 
## Tecnologias utilizadas 
- HTML
- CSS
- JavaScript
- Docker

--- 

## Executando a aplicação com Docker 

### 1. Pré-requisitos 
Certifique-se de ter instalado em sua máquina: 
- Docker
- Docker compose

### 2. Clonando o repositório
```bash
git clone https://github.com/mariaandrezacs/desafio-projeto-docker.git
cd desafio-projeto-docker
```

### 3. Executanto o projeto
```bash
docker compose up | docker-compose up
```
